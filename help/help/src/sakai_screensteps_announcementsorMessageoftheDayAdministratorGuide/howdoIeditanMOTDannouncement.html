How do I edit an MOTD announcement?

Go to Announcements.

Select the Announcements tool from the Tool Menu of the Administration Workspace.

Click Edit.

Click the Edit link below the announcement you want to modify.

Edit the announcement.

(Optional) View Revision History.

If you would like to see information about previous revisions of this announcement, click the See revision history link.

Click Save Changes.

When you have finished your edits, click the Save Changes button to save your work.
