How do I add a syllabus item to the calendar?

Once you have added items to the Syllabus tool, you may add start and end times to them for informational purposes. The dates specified for an item do not control visibility of the item. However, the dates do control the display of items in the Calendar tool.

You must specify a start and/or end time for the item before you can add it to the calendar.

A syllabus item must be published before you can add it to the calendar.

Go to Syllabus.

Select the Syllabus tool from the Tool Menu in your site.

Select Manage.

Specify a Start Time and End Time for the item/s (if you have not already done so).

Check the box in the Calendar column for the item/s you would like to add to the calendar.

Remember that the item must be published first, before you can add it to the calendar.

To remove an item from the calendar, deselect the check box.

Select Update to save your changes.

You may also add items to the Calendar from the Edit screen.

Check the box for Display dates on calendar to add the item to the calendar.
